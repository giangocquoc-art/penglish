# GSC Optimization Rules

Sau khi deploy 14-30 ngày, export từ Google Search Console:
Performance -> Search results -> Queries -> Export CSV.

Đặt file tại:
docs/seo-90-days/gsc/gsc_queries.csv

Các cột nên có:
Query,Clicks,Impressions,CTR,Position

## Luật tối ưu

1. Position 1-3:
- Giữ title/meta.
- Thêm update nhỏ.
- Thêm internal link từ trang liên quan.

2. Position 4-10:
- Tối ưu title tăng CTR.
- Thêm FAQ chứa query thật.
- Thêm bảng/bài tập nhỏ.

3. Position 11-20:
- Thêm content block 300-600 chữ.
- Thêm internal link từ pillar page.
- Thêm schema nếu thiếu.

4. Position 21-50:
- Kiểm tra intent.
- Có thể tách thành trang riêng.
- Đổi H2/H3 để khớp query hơn.

5. Impression cao CTR thấp:
- Viết lại title/meta.
- Thêm năm/current wording nếu phù hợp.
- Title phải rõ lợi ích, không chỉ nhồi keyword.
