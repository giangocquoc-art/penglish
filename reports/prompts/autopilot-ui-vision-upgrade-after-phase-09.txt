CORE PRODUCT VISION:
P-English should feel like a polished Vietnamese-first English learning web app with:
- bright ocean background
- soft sky / cloud / underwater atmosphere
- glassmorphism learning widgets
- a friendly whale mascot as the product symbol
- subtle whale/ocean motion using GSAP where practical
- clear CEFR learning journey
- confidence-building progress UI
- mobile-first layout similar to a real learning app
- clean Vietnamese explanations
- English only where learners practice English
- no clutter
- no noisy repo/license text in learner UI

WHALE VISION:
The whale is not decoration only.
It should act as a calm learning companion:
- subtly swims in the background
- never chases or sticks too close to the cursor
- never covers text/buttons
- gives meaningful coaching based on progress/next unit/weak skill
- streak can make the whale grow slightly
- after a reasonable size threshold, small baby whales may appear
- whale size must be capped so it never blocks the UI
- respect prefers-reduced-motion
- clean up GSAP timelines on unmount
