# Product Context

P-English is a Vietnamese-first English learning web app.

Learners should feel:
- The app is friendly and easy to use.
- Lessons are clear, practical, and not overwhelming.
- Progress is visible.
- Keyboard interactions are smooth.
- The whale/ocean identity supports learning without clutter.

The lesson system should support long-term growth:
- More CEFR levels.
- More vocabulary.
- Grammar points.
- Flashcards.
- Guided lessons.
- Shadowing/video practice later.
