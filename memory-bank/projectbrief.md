# Project Brief: P-English Lesson Import

P-English needs a safe, repeatable system to grow English lessons from downloaded open-source repositories.

The system should not rely on Cline inventing lessons freely. It should inspect downloaded sources, review licenses, extract safe content, normalize lessons into a strict P-English schema, validate output, then integrate the result into the existing P-English UI.

Primary goal:
Create a scalable lesson import pipeline for A1, A2, and B1 lessons with Vietnamese learner-friendly explanations.
